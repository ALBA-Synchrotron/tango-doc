Tools and libraries
-------------------
Tango Controls provides various tools. You may refer to :doc:`Index of tools <tools/index>` for a detail list
of available tools, applications and libraries or look below for the most used ones.

* :doc:`Astor <tools/astor/index>` which is a tool for management of Tango Controls system.
* :doc:`Jive <tools/jive/index>` which is used to configure components of the Tango Controls and brows a static
  :term:`Tango database`.
* :doc:`Pogo <tools/pogo/index>` which is a :term:`device class`es/:term:`device servers` generator.
* :doc:`JDraw and Synpotic <tools/jdraw/index>` which are a synoptic panels builder and a viewer.
